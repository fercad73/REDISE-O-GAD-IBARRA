# GAD Ibarra - Portal Responsivo

## Descripción
Rediseño responsivo completo del portal del GAD Municipal de Ibarra, implementando las mejores prácticas de desarrollo web moderno.

## Características
- ✅ Diseño 100% responsivo (Mobile First)
- ✅ Modo claro/oscuro con persistencia
- ✅ Accesibilidad WCAG 2.1 AA
- ✅ PWA (Progressive Web App)
- ✅ Offline support
- ✅ Chat de asistencia virtual
- ✅ Carrusel de imágenes
- ✅ Estadísticas animadas
- ✅ Formularios con validación
- ✅ Notificaciones toast
- ✅ SEO optimizado
- ✅ Performance optimizada


## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript ES6+
- Font Awesome 6
- Service Workers

## Instalación
1. Clonar el repositorio
2. Abrir index.html en un servidor local
3. Para PWA: servir con HTTPS

## Validación
- HTML validado con W3C Validator
- CSS validado con W3C CSS Validator
- Performance: Lighthouse 90+ puntos
- Accesibilidad: Wave Tool sin errores críticos
